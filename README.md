# yapi to type chrome extension


generate type for yapi docs

## Prerequisites

* [node + npm](https://nodejs.org/)

## Build
run
```
npm i
npm run build
```
## Load extension to chrome

Load `dist` directory

## usage

just click the extension icon
the popup page will show the type immediately
